# 기획

다양한 내기를 하며 도파민을 충전하는 당신들 통계를 만들어드립니다.
(돈 = 도파민 비슷한 느낌이지만 돈으로만 내기를 하는것은 아니니까 도파민이라는 단위를 사용)

# 기능

## 도파민 스테이션

1. 도파민 스테이션 생성기능
   도파민 스테이션은 (그룹과 비슷한 느낌임)

2. 도파민 스테이션 초대기능

3. 도파민 스테이션 리스트
   유저는 초대받은 스테이션에 접근할 수 있다.

4. 도파민 스테이션에는 프로필사진, 제목, 설명, 인원수가 노출됨

## 도파민 스테이션 상세

1. 유저들별로 패배,승리 수, 손해본 금액, 스테이션안에서 사용된 총 금액 대시보드

2. 다양한 내기 게임 리스트

- 악어이빨 게임
- 그냥 랜덤뽑기 게임
- 유명한 공이 떨어지면서 승리자 패배자 고르는 게임
  좋은 내기 게임 있으면 더 다양하게 AI께서 추천해주시오

3. 손해/이득 도파민 직접입력하는 기능

## 스테이션 > 유저 상세

1. 대시보드 상세 노출

2. 카드 공유하기
   카드에는 주요 대시보드 내용 노출
   손해보고 있으면 운이 없는 쓰레기 등 강력한 표현으로 놀림
   이득보고 있으면 주변에서 꽁짜로 도파민 주입시켜주는 사람 등 주변을 놀리기 좋은 멘트
